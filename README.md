# Lab_21_dz
Домашняя работа к лабораторной работе №21

Проект разработан в учебных целях и демонстрирует практическое использование:

- структур данных (`struct`)
- массивов структур
- указателей
- стандартных библиотечных функций
- алгоритмов поиска и сортировки

Программа позволяет работать с набором данных о карнизах, выполнять поиск по материалу и сортировать записи по площади.

---

##  Функциональные возможности

### Основной функционал

-  Хранение данных в массиве структур
-  Вывод всех записей в табличном виде
-  Поиск карниза по материалу
-  Сортировка карнизов по площади (длина × ширина)
-  Использование стандартной функции `qsort`
-  Консольный ввод и вывод данных

---

##  Структура данных

Каждый карниз описывается структурой `cornice_t`:

| Поле | Тип | Описание |
|----|----|---------|
| `length` | `int` | Длина карниза |
| `ceilingWidth` | `int` | Ширина по потолку |
| `wallHeight` | `int` | Высота по стене |
| `material` | `char[20]` | Материал карниза |

##  Алгоритмы

### Поиск

- Реализован **линейный поиск**
- Выполняется по строковому полю `material`
- Используется функция `strcmp`
- Возвращает указатель на найденный элемент или `NULL`

### Сортировка

- Используется стандартная функция `qsort`
- Сортировка производится по площади карниза:
  
площадь = длина × ширина по потолку
